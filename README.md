# Moja Strona Full-Stack

Aplikacja webowa full-stack z własnym backendem w Node.js i bazą danych PostgreSQL.  
Projekt obejmuje:  
- logowanie i rejestracja użytkowników  
- zarządzanie sesjami i ciasteczkami (`pg-session`)  
- walidację danych i szyfrowanie haseł (Argon2)  
- frontend w HTML, CSS i JavaScript  

## Technologie
Node.js, Express, PostgreSQL, HTML, CSS, JavaScript, Argon2, pg-session
